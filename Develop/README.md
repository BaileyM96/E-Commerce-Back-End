# E-Commerce Back End



## Description

My goal is to build the back end for a exsisting E-Commerce website I have to configure express.js API and sequelize to interact with the mySQL database.

## Technology Used

Express.js,
MySQL2,
Sequelize,
dotenv,
JavaScript

## Installation

Clone it from the github repo
